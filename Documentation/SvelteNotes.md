A svelte file should consist of 3 sections
**The script section**
```
<script lang="ts">

</script>
```
**The HTML section**
Note each svelte component should have one root html tag. Most likely a div.
```
<div>
<!-- Code here -->
</div>
```
**The CSS Section**
```
<style>
.class {
    font-size: 2em;
}
</style>
```

An additional section that may be used, if needed
**svelte:head**
```
<svelte:head>
<!-- Code here -->
</svelte:head>
```
